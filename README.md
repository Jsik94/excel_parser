## 🛠How to use🛠

1. excel_parser 하위에 dataset이라는 폴더를 만들어 줍니다.
2. KOSMO 형식의 모든 구인카드를 dataset 폴더에 넣어줍니다.
3. excel_parser.bat 파일을 클릭하여 실행합니다.
4. 결과는 dist폴더에 result.csv로 저장됩니다.
